Useful Information

Common Prefix Lengths

  • /48: Site prefix (ISP allocation)
  • /56: Home network prefix
  • /64: Subnet prefix (standard)
  • /128: Host address

Special Addresses

  • ::1/128: Loopback address
  • ::/128: Unspecified address
  • fe80::/10: Link-local addresses
  • fc00::/7: Unique local addresses
